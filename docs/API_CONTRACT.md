# API Contract

공통 API 문서가 아직 연결되지 않았습니다.

## AI 필수 규칙

- 실제 API 연동은 이 문서에 공식 API 문서 URL 또는 endpoint 명세가 추가된 뒤에만 진행합니다.
- 이 문서에 없는 endpoint, field, status, 인증 방식을 추측해서 만들지 않습니다.
- API 문서가 없을 때는 mock 데이터 기반 화면 시안만 만들 수 있습니다.
- 실제 회원, 상품, 주문, 결제, 개인정보, 권한, 파일 업로드 연동은 개발자 확인 전까지 구현하지 않습니다.
